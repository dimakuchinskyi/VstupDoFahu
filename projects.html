Студентські проєкти

Реальні VR/AR додатки, створені нашими студентами

Чому проєкти важливі?

На кафедрі ми віримо, що найкращий спосіб навчитися — це створювати. Студенти працюють над реальними проєктами протягом усього навчання, від невеликих лабораторних до повноцінних дипломних робіт.

Флагманські проєкти

Найуспішніші та найцікавіші проєкти наших студентів

Mozilla Hubs

VR · Web · Open Source

Опис: Відкрита платформа для створення віртуальних кімнат і спільної роботи у VR прямо в браузері.

Спробувати GitHub

Tilt Brush (Open Brush)

VR · Art · Open Source

Опис: Відомий VR-додаток для малювання у 3D-просторі. Відкритий код, підтримка багатьох VR-платформ.

Демо-відео GitHub

AR.js

AR · Web · Open Source

Опис: Фреймворк для створення AR-додатків прямо у браузері. Підтримує marker-based та location-based AR.

Документація GitHub

Sketchfab

3D · AR/VR · Viewer

Опис: Платформа для перегляду та публікації 3D-моделей з підтримкою AR/VR. Доступна онлайн та через мобільні додатки.

Сайт Демо-відео
